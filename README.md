# Commandline-Calculator
A simple command line calculator

# How to install
Windows: <br>
Compile with Visual Studio

Linux: <br>
Compile with CMake

# Using cmdcalc
- Expressions (<code>1 + 2 - 3 * a ** (42 / 2)</code>)
- Variables (<code>a=99</code>)

# Operators
- + Addition
- - Subtraction
- * Multiplication
- ** Power
- / Division
- = Assignment
